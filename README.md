# Image-analysis

# 🔥 Wildfire Image Analysis using Gemini AI

This project uses **Google Gemini AI** to analyze images and provide insights about **wildfire satellite images**.
It includes:
* 🌐 Streamlit Web App
* 💻 Python CLI Version

✨ Features

* 📷 Upload or load an image
* ❓ Ask questions about the image
* 🔥 Detect wildfire-related content
* 📊 Get insights like:

  * Burn area estimation
  * Fire spread analysis
  * Causes & prevention tips

🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* PIL

🚀 How to Run

▶️ Install Requirements

```bash
pip install streamlit google-generativeai pillow
```

🌐 Run Web App

```bash
streamlit run streamlit_app.py
```

💻 Run CLI Version

```bash
python cli_script.py
```
